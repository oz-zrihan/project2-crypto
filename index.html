Hi I’m Oz zrihan, Graphic designer and ui/ux expert by day And Full-stack enthusiast by night.
I finished my bachelor degree in Mass media and digital communication with excellence. I got more the ten years of experience in web design and ui/ux, And now it's the time to go up a level, so I took a full-stack curse at “John Bryce”.

This project meant to show my level at the middle of the curse and to demonstrate the use of Ajax and jQuery.
In this project I made an Ajax call using jQuery Ajax in order to get the first 100 most popular digital coins. Every coin got is own Ajax call to get more information about it, including the coin summery, the coin value in shekel dollar and euro, and the last week candle-stick chart. Also the user can add up to five coin to his “watch list” so he could compare between the coins and get real time information about then at “live reports” page.

As a graphic designer and ui/ux, I think a lot about the user experience, even when I do something like “home project” so it was important to me to add the full screen header with mouse animation and parallax scrolling, also to add the loading animation at the “live reports” page in order to give the chart some time to get its first data call so the user won’t see empty chart at the beginning.

Hope you enjoy this project
